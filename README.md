# GymBeam Case Study Intern

Interaktívna Streamlit aplikácia pripravená ako prezentácia case study a dotazníka.

## Obsah projektu

Projekt obsahuje:
- sekciu **Zadanie** (5 otázok case study)
- sekciu **Dotazník** (20 osobných otázok)
- export celého obsahu do **PDF**
- jednoduché webové rozhranie vytvorené v **Streamlit**
